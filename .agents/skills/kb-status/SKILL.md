# /kb-status

## Run

**Helper:** `.agents/skills/kb-status/status.py`

```bash
uv run .agents/skills/kb-status/status.py
```

Read-only. Reads cached `wiki/overview.md` (regenerated by `/kb-lint`). If `overview.md` is missing or stale, run `/kb-lint` first.

## Tests

```bash
uv run --with pytest --with ruamel.yaml pytest .agents/skills/kb-status/tests/test_status.py -v
```

**Purpose:** Dashboard — reads cached wiki/overview.md; does not regenerate it. Reports inbox depth, quarantine depth, last ingest per channel, lint debt, open threads, log size, _inbox/edges.md backlog.

**Reads:** wiki/overview.md, wiki/log.md, raw/inbox/, raw/quarantine/, wiki/_inbox/edges.md
**Writes:** stdout only
**Idempotency:** read-only; reads cached `wiki/overview.md`.

## Output format

```
work-brain status — YYYY-MM-DD HH:MM

Pages:        Person=N  Org=N  Project=N  Topic=N  Decision=N  Meeting=N  Source=N  Artifact=N  Event=N
Inbox:        journal=N  meetings=N  emails=N  chats=N  clippings=N  docs=N
Quarantine:   N files
Edges inbox:  N proposals  (rotation threshold: 500)
Log:          N entries  (rotation threshold: 1000)
Last ingest:  journal=YYYY-MM-DD  meetings=YYYY-MM-DD  ...
Open threads: N
```

## Manual checklist

- [ ] Read wiki/overview.md for page counts and last ingest timestamps
- [ ] Count files in each raw/inbox/<channel>/ subdirectory
- [ ] Count files in raw/quarantine/
- [ ] Count entries in wiki/_inbox/edges.md (lines starting with "- (")
- [ ] Count entries in wiki/log.md (lines starting with "## [")
- [ ] Count wiki/threads/ files
- [ ] Print dashboard to stdout
